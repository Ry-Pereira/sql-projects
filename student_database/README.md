# 🗄️🎓 Student Information Database Project 🚀💻

## 📌 Overview 🧠

This project contains a basic SQL database setup for managing student information 📊📚💾.

The script creates a database named `studentInformation` 🗄️, defines a `student` table 🧱 with common student attributes 👤, modifies the table structure ⚙️, and retrieves stored data using SQL queries 🔍.

SQL 🗄️ is the primary language used throughout this project 🌐.

---

### 🎯 Goals of this Project 🚀

* 🧹 Practice database creation and management  
* 📈 Demonstrate understanding of SQL table structures  
* 🧭 Learn how to modify existing database tables  
* ⚙️ Showcase basic SQL querying and data organization  
* 🧠 Strengthen database design and problem-solving skills  

---

## 🧱 Project Structure 📁

This project includes:

* 🗄️ Database creation  
* 📋 Table creation  
* ➕ Table modification using `ALTER TABLE`  
* 🔍 Data retrieval using `SELECT` statements  
* 📊 Student data organization and management
